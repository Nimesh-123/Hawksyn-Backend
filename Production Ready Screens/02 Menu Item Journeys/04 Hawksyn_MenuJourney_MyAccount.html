Colour dots
Amber dot = CV/data event. Green dot = payment or enable event. Blue dot = login event. Small visual tag system — scannable at a glance without reading every row.
Newest first
Reverse chronological — most recent at top. User almost always wants to check "what just happened." Oldest entries scroll down as far as account creation date.
S04-E
Change M-PIN — current PIN → new PIN → confirm
Security
PIN change flow
Three steps inline on one screen: enter current PIN, enter new PIN, confirm new PIN. No navigation between steps — they appear sequentially as user completes each. Simple and fast.
Micro: wrong PIN
If current PIN is wrong: dots shake, turn red, clear. "Incorrect PIN" message below. 3 wrong attempts → lockout screen (same as J4-S04). Consistent security behaviour everywhere.
