Main
Four toggles
Notifications on/off. HIP visible on/off. WhatsApp communication on/off. Each toggle is independent. Delete account at bottom — in a separate danger zone card, not inline with the toggles.
Toggle design

S05-C
HIP toggled OFF — inline state change, no navigation
HIP off
Micro moment
User taps HIP toggle. Toggle flips to OFF instantly. Row background shifts to amber tint. Sub-text changes to "Your HIP link is now paused." Small amber warning tag appears. No screen change.
Immediate effect
No confirmation needed for toggling HIP off — it is reversible. User sees the consequence in the same moment they act. Toggle back on to restore. Clear, low-friction.

S05-D
Delete account — confirmation sheet, type to confirm
Delete
Hard confirmation
Tapping "Delete my account" raises a bottom sheet. User must type "DELETE" to unlock the final button. Button stays disabled (red, dimmed) until exact text entered. Irreversible action requires friction.
What gets deleted
Sheet lists exactly what will be removed: profile, scan history, HIP, all downloads. No surprises. User knows the full cost before confirming. 30-day recovery window noted.
