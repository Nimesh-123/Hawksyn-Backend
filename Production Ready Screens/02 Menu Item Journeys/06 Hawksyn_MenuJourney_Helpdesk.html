Settings
No back arrow
Top bar has no back arrow on this screen — user arrived here by submitting a form. Back would re-submit. "Back to tickets" is the only navigation. Prevents accidental double-submit.
Ticket ID tap
"Tap to copy" hint below the ID. Single tap copies to clipboard. Brief green tick flash on the ID text confirms copy. User can paste it in email or share with others.
S06-E
Ticket thread — conversation view, open ticket
Thread
Thread view
Tapping any ticket from the list pushes to this thread view. User message on right (amber). Support reply on left (dark card). Ticket status + ID at top. Read-only when Closed.
Reply bar
For Open tickets: input bar at bottom — type + send. For Closed: input bar replaced with "This ticket is closed" muted note. User cannot reopen — must raise a new ticket.
