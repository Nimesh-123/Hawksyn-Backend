STATE TRANSITION RULES
GREEN → AMBER
Triggered when remaining window drops below 30%. Slow pulse animation starts. No push notification. No alert screen.
AMBER → RED
Triggered when remaining window drops below 15%. Fast pulse + RISK badge. Push notification fires once. Full-screen alert on next app open.
RED → BLUE
Clock hits zero (window expired). Ring turns solid blue. Refresh button greyed out until admin unlocks. Value shows 0 or "EXPIRED".
BLUE → GREEN (reset)
User taps refresh after admin unlock. New audit window starts. Clock resets to full countdown. Ring transitions back to green. Push notification fires on admin unlock.
RING DEPLETION ANIMATION SPEC
→ Ring depletes clockwise from 12 o'clock position (top)
→ Full ring = full audit window remaining
→ Ring gap shows proportional days elapsed vs total window
→ Depletion is NOT real-time — ring updates only on app open
→ Ring colour transitions smoothly: Green → Amber → Red as thresholds are crossed
→ Transition animation duration: 600ms ease-in-out
→ Ring stroke width: 8px on 90×90px clock
→ Background circle always shows the dark tint version of the ring colour
